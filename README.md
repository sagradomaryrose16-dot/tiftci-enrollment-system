# TIFTCI Enrollment System

## Description
TIFTCI Enrollment System is a web-based application developed to streamline enrollment and academic data management for an educational institution.

## Features
- Admin dashboard
- Student & faculty management (CRUD)
- Academic year and level/section setup
- Session-based authentication
- Relational database structure

## Technologies Used
- PHP
- MySQL / phpMyAdmin
- HTML5
- CSS
- XAMPP / Localhost

## Installation
1. Install XAMPP
2. Copy the project folder to `xampp/htdocs`
3. Import `tiftci_enrollment_db.sql` in phpMyAdmin
4. Open browser and run:
   `http://localhost/TIFTCI ENROLLMENT SYSTEM/`
